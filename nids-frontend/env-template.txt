# Setup Instructions:
# 1. Copy this file and rename it to .env.local
# 2. Go to https://clerk.com and create a free account
# 3. Create a new application
# 4. In your Clerk dashboard, go to "API Keys" section
# 5. Copy your publishable key (starts with pk_test_) and secret key (starts with sk_test_)
# 6. Replace the placeholder values above with your actual keys
# 7. Save the .env.local file and restart your development server
# 8. Test authentication at http://localhost:3000/test-auth

# Troubleshooting:
# - If authentication doesn't work, check that your keys are correct
# - Make sure there are no extra spaces or quotes around the keys
# - Restart your development server after adding the keys
# - Check the browser console for any error messages
